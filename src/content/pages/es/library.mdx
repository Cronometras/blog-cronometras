  <p class="mb-6 text-gray-700 dark:text-gray-300">
    La <strong>Biblioteca de Elementos</strong> es una funcionalidad central de <a href="/es/features" class="text-accent hover:underline">CRONOMETRAS</a> que permite a los usuarios buscar, seleccionar y reutilizar elementos de trabajo previamente creados en diferentes <a href="/es/blog/mejores-practicas-metodo-trabajo" class="text-accent hover:underline">estudios de métodos y tiempos</a>. Esta herramienta facilita la consistencia entre estudios y mejora significativamente la eficiencia del proceso de creación de nuevos análisis, contribuyendo a un <a href="/es/blog/importancia-del-calculo-del-tiempo-estandar-para-la-productividad-y-rentabilidad" class="text-accent hover:underline">cálculo preciso del tiempo estándar</a>.

*   **Búsqueda en Tiempo Real:** Campo de búsqueda con tecnología debounce que filtra elementos por nombre, descripción o estudio al que pertenecen, optimizando el tiempo de respuesta.
*   **Visualización en Tarjetas:** Resultados mostrados en un formato de tarjetas intuitivo con información clave sobre cada elemento, facilitando la identificación visual.
*   **Acceso a Estudios Compartidos:** Visualización de elementos provenientes tanto de estudios propios como de organizaciones a las que pertenece el usuario, fomentando la colaboración.
*   **Información Detallada:** Cada tarjeta muestra el tipo de elemento, frecuencia, tiempo observado y suplementos asignados para una evaluación rápida.

*   **Origen de los Elementos:** Cada elemento muestra el estudio de origen para mejor contexto y trazabilidad de los datos.
*   **Gestión de Selección:** Opción para limpiar toda la selección y comenzar de nuevo, facilitando la experimentación con diferentes combinaciones.

*   **Formulario Modal Intuitivo:** Interfaz clara para ingresar información básica del nuevo estudio, incluyendo nombre, empresa, fecha y escalas de actividad.
*   **Preservación de Datos Históricos:** Mantenimiento automático de los suplementos, registros de tiempo y configuraciones de los elementos originales.
*   **Validación Integrada:** Verificación automática de que se hayan completado todos los campos requeridos y que haya elementos seleccionados.
*   **Transición Fluida:** Redirección automática al nuevo estudio creado, permitiendo comenzar inmediatamente con el análisis.

<Section
  section={{
    title: "Promediado de Elementos para Estandarización de Tiempos",

*   **Pestaña Dedicada:** Sección específica para el cálculo de promedios entre elementos similares, separando claramente esta funcionalidad.
*   **Cálculo Automático Inteligente:** Generación de un nuevo elemento que integra:
    *   Tiempo promedio calculado a partir de todos los elementos seleccionados
    *   Suplementos promediados según la configuración de cada elemento
    *   Descripción detallada que documenta los elementos de origen para trazabilidad
*   **Visualización de Resultados:** Panel que muestra el elemento promediado con todos sus parámetros antes de su incorporación a la biblioteca.
*   **Integración Perfecta:** Opción para enviar el elemento promediado directamente a la selección en la biblioteca principal.

  section={{
    title: "Flujos de Trabajo Optimizados para Estudios de Métodos y Tiempos",

*   **Flujo Simple:** Selección directa de elementos existentes y creación inmediata de un nuevo estudio, ideal para análisis rápidos.
*   **Flujo con Promedio:** Creación de elementos promediados a partir de elementos similares antes de incorporarlos al nuevo estudio, perfecto para estandarización.
*   **Flujo Mixto:** Combinación de elementos individuales y promediados en un mismo estudio, ofreciendo máxima flexibilidad.
*   **Selecciones Independientes:** Mantenimiento de selecciones separadas en las pestañas de biblioteca y promedio, permitiendo trabajar en paralelo en diferentes aspectos.
