Un cronómetro es un dispositivo utilizado para medir el tiempo transcurrido. 

Existen muchos tipos diferentes de cronómetros, cada uno diseñado para medir y registrar el tiempo de manera diferente. Algunos de estos tipos incluyen cronómetros mecánicos, electrónicos, aplicaciones para dispositivos móviles y de computadora. 

## Los cronómetros mecánicos 

son los más antiguos de los cronómetros y funcionan mediante un mecanismo de engranajes. Estos dispositivos se utilizan para medir el tiempo que se tarda en completar una tarea o una actividad, como una carrera. El cronómetro mecánico más antiguo conocido se remonta al año 1556. 

La Oficina Internacional del Trabajo recomienda para efectos del estudio de tiempos dos tipos de cronómetros: mecánico y electrónico. El cronómetro mecánico a su vez puede subdividirse en ordinario, vuelta a cero y cronómetro de registro fraccional de segundos

Un cronómetro mecánico ordinario es un reloj de precisión que se utiliza para medir tiempos mediante la puesta en marcha y parada del mecanismo de control que generalmente son botones que ponen en marcha el inicio del tiempo o paran la medición de éste en el momento que se acciona

Un cronómetro mecánico vuelta a cero funciona de manera que al acabar cada elemento se hace volver el reloj a cero y se lo pone de nuevo en marcha inmediatamente para cronometrar el elemento siguiente . Esto permite medir el tiempo de cada elemento de manera individual sin tener que detener el mecanismo del reloj.

Un cronómetro de registro fraccional de segundos es un tipo de cronómetro mecánico que permite medir fracciones de segundo. Este tipo de cronómetro se utiliza en estudios de tiempos y puede ser configurado para mostrar diferentes unidades de tiempo como segundos (MKS), minutos decimales (1/100 minutos) o horas decimales .

## Los cronómetros electrónicos 

son dispositivos modernos que se utilizan para medir el tiempo. Estos dispositivos generalmente se componen de una pantalla digital, un botón de encendido / apagado y una batería para alimentar el dispositivo. Los cronómetros electrónicos se pueden usar para medir y registrar los tiempos de varias actividades, desde una carrera hasta una partida de tenis y como no, también los procesos industriales.

## Las avanzadas aplicaciones de cronometraje

Aplicaciones de cronometraje como cronometras.com son [aplicaciones para el cronometraje de tiempos de trabajo](https://cronometras.com) muy avanzadas que permiten medir tiempos de diferentes elementos de trabajo tanto de manera escalonada, como elemento por elemento.

Tienen la ventaja de poder instalarse en dispositivos móviles y tablets lo que permite una gran movilidad del técnico que realiza el estudio.

Son soluciones asequibles para cualquier tamaño de empresa dado su bajo costo y muy flexibles y adaptables a diferentes entornos de trabajo.

## Los cronómetros de computadora 

son dispositivos electrónicos que se conectan a una computadora para medir el tiempo. Estos dispositivos se usan para medir el tiempo de carreras de velocidad, así como para medir el tiempo que se tarda en completar una tarea o una actividad. Estos dispositivos son capaces de medir con gran precisión el tiempo transcurrido.  Algunos cronómetros de computadora están diseñados para el estudio de tiempos en la industria.

Los cronómetros de computadora son programas que se ejecutan en una computadora y permiten medir el tiempo de manera precisa. Estos cronómetros pueden ser descargados como software o utilizados en línea. Al igual que los cronómetros mecánicos, los cronómetros de computadora pueden ser iniciados y detenidos a voluntad del usuario y permiten marcar vueltas o intervalos mientras el cronómetro está en marcha 

En resumen, existen muchos tipos diferentes de cronómetros, cada uno diseñado para medir y registrar el tiempo de manera diferente. Los cronómetros mecánicos son los más antiguos, mientras que los cronómetros electrónicos, magnéticos y de computadora son los más modernos. Estos dispositivos se utilizan para medir el tiempo de carreras de velocidad, así como para medir el tiempo que se tarda en completar una tarea o una actividad.
