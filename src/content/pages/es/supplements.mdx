  <h2 class="text-2xl font-bold mb-4 text-gray-800 dark:text-white">¿Qué es el Módulo de Suplementos en CRONOMETRAS?</h2>

  <p class="mb-6 text-gray-700 dark:text-gray-300">
    El <strong>Módulo de Suplementos</strong> es una herramienta especializada dentro de <a href="/es/features" class="text-accent hover:underline">CRONOMETRAS</a> diseñada para calcular y aplicar los tiempos adicionales necesarios a los elementos de trabajo. Estos suplementos compensan la fatiga, necesidades personales y condiciones ambientales, permitiendo obtener un tiempo estándar justo y realista para los trabajadores.
  </p>
</div>

<Section
  section={{
    title: "Selección de Sistema de Suplementos para Estudios de Tiempos Precisos",
    content:
      "El <strong>cronometraje industrial</strong> profesional requiere sistemas estandarizados para la asignación de suplementos:",
    image: "/images/fatigue-allowances.png",
    imageClasses: "w-64 h-auto object-contain mx-auto section-image-hover",
    alt: "Sistemas de suplementos en software de cronometraje industrial"

/>

*   **Tablas OIT (versión 4):** Sistema basado en los estándares de la Organización Internacional del Trabajo, reconocido mundialmente para la asignación de suplementos en estudios de tiempos.
*   **Tablas TAL:** Sistema alternativo basado en las tablas de la OIT pero con factores más objetivados, utilizando mediciones precisas como ruido en decibelios e iluminación en luxes.
*   **Cambio Dinámico:** Posibilidad de alternar entre ambos sistemas según las necesidades específicas de cada estudio, con alertas de compatibilidad para evitar inconsistencias.

<Section
  section={{
    title: "Gestión de Elementos para Cálculo Preciso del Tiempo Estándar",
    content:
      "Una <strong>medición del trabajo</strong> efectiva requiere una gestión clara de todos los elementos y sus suplementos:",
    image: "/images/personal-needs.png",

*   **Visualización Completa:** Lista detallada de todos los elementos del método con su descripción, tipo y porcentaje de suplemento asignado.
*   **Acciones Rápidas:** Funcionalidades para copiar suplementos entre elementos similares, ahorrando tiempo en la configuración.
*   **Reinicio Selectivo:** Opción para restablecer los valores de suplementos asignados a elementos específicos o a todo el estudio.
*   **Indicadores Visuales:** Representación gráfica del porcentaje de suplemento aplicado a cada elemento para una rápida evaluación.

<Section
  section={{
    title: "Asignación de Suplementos por Puntos para Estudios de Tiempos Objetivos",
    content:
      "El <strong>software de cronometraje industrial</strong> CRONOMETRAS implementa un sistema de puntos estructurado para la asignación de suplementos:",
    image: "/images/work-conditions.png",
    imageClasses: "w-64 h-auto object-contain mx-auto section-image-hover",
    alt: "Sistema de puntos para suplementos en estudios de métodos y tiempos"
  }}
/>

*   **Categorías Estructuradas:** Los factores se organizan en tres categorías principales:
    *   **Categoría A:** Esfuerzo físico (uso de fuerza, postura, vibraciones, ciclo breve, ropa incrómoda)
    *   **Categoría B:** Esfuerzo mental (concentración, monotonía, tensión visual, ruido, tensión mental, tedio)
    *   **Categoría C:** Condiciones de trabajo (temperatura, ventilación, emanaciones de gases, polvo, suciedad, presencia de agua)
*   **Tablas Interactivas:** Interfaces intuitivas para seleccionar valores específicos para cada factor según su intensidad.
*   **Cálculo Automático:** Conversión instantánea de los puntos asignados a porcentajes de suplemento según algoritmos estandarizados.
*   **Interfaces Especializadas:** Herramientas visuales adaptadas para factores complejos como uso de fuerza, postura y condiciones ambientales.

<Section
  section={{

    content:

*   **Asignación Directa:** Posibilidad de establecer manualmente un porcentaje de suplemento sin utilizar el sistema de puntos.
*   **Anulación Selectiva:** El suplemento forzado anula cualquier cálculo previo basado en puntos, manteniendo la flexibilidad del sistema.
*   **Aplicación Específica:** Capacidad para aplicar suplementos forzados a elementos individuales sin afectar al resto del estudio.
*   **Documentación Automática:** El sistema registra que el suplemento ha sido asignado manualmente para referencia y auditoría futura.

<Section
  section={{
    title: "Cálculo de Fatiga para Máquinas: Optimización de Tiempos Concurrentes",
    content:
      "La <strong>herramienta de cálculo de tiempo estándar</strong> incluye funcionalidades especializadas para operaciones con maquinaria:",
    image: "/images/machine-fatigue.png",
    imageClasses: "w-64 h-auto object-contain mx-auto section-image-hover",
    alt: "Cálculo de fatiga para máquinas en estudios de tiempos"

/>

*   **Análisis Completo:** Evaluación detallada de los tiempos de máquina en funcionamiento, parada y tiempo concurrente.
*   **Fórmulas Adaptativas:** Aplicación de diferentes algoritmos según el tiempo de inactividad del operario:
    *   ≤30 segundos: Fórmula para inactividad muy breve
    *   31-90 segundos: Fórmula para inactividad breve
    *   91-600 segundos: Fórmula para inactividad media
    *   >600 segundos: Fórmula para inactividad prolongada
*   **Suplementos Ponderados:** Cálculo de suplementos basados en el peso relativo de cada elemento en el ciclo total de trabajo.
*   **Explicación Detallada:** Documentación paso a paso del proceso de cálculo realizado para facilitar la comprensión y validación.
*   **Normalización Automática:** Ajuste de tiempos según la actividad registrada y la frecuencia de cada elemento para obtener resultados precisos.
